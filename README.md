# 📋 JOGO DA ORGANIZAÇÃO

Sistema desktop desenvolvido em **Java + Swing** com foco em **gamificação de hábitos e organização pessoal**.

O projeto transforma atividades do dia a dia em um sistema de progresso semelhante a jogos, onde o usuário ganha **XP**, sobe de **nível** e acompanha sua evolução conforme realiza hábitos positivos.

---

# 🚀 SOBRE O PROJETO

O **Jogo da Organização** foi criado com objetivo de praticar:

* Programação Orientada a Objetos (POO)
* Interfaces gráficas com Java Swing
* Manipulação de tabelas e eventos
* Uso de ENUMs
* Estruturação em camadas

A ideia central é aplicar conceitos de produtividade pessoal usando mecânicas de jogos.

---

# 🎮 FUNCIONALIDADES

✅ Cadastro de hábitos em tabela
✅ Seleção de atividades realizadas
✅ Sistema de ganho de XP por hábito
✅ Barra de progresso visual
✅ Sistema de Level Up automático
✅ Atualização dinâmica de nível na interface

---

# 🧠 MECÂNICA DE PROGRESSÃO

Cada hábito possui um valor de XP definido via ENUM.

Exemplo:

* Academia → +150 XP
* Estudar → +300 XP
* Orar → +200 XP

Ao atingir a quantidade necessária:

✅ o jogador sobe de nível
✅ a barra de progresso é atualizada
✅ o próximo nível exige mais XP
